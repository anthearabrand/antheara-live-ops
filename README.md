# Antheara Live Ops

ระบบจัดการทีมไลฟ์ขายของ Antheara (TikTok + Shopee) — ตารางไลฟ์ ยอดขาย สต๊อก และเงินเดือน/ค่าคอม

- `index.html` — แอพจริง เปิดใช้งาน
- `showcase.html` — หน้าโบรชัวร์แนะนำระบบ
- `supabase-schema.sql` — โครงฐานข้อมูล (รันแล้วบน Supabase)

ข้อมูลเก็บอยู่บน Supabase (ฐานข้อมูลกลาง) ทุกคนที่ล็อกอินจะเห็นข้อมูลร่วมกันแบบเรียลไทม์
