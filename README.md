# Antheara Live Ops

ระบบจัดการทีมไลฟ์ขายของ Antheara (TikTok + Shopee) — ตารางไลฟ์ ยอดขาย สต๊อก และเงินเดือน/ค่าคอม

- `index.html` — แอพจริง เปิดใช้งาน
- `showcase.html` — หน้าโบรชัวร์แนะนำระบบ
- `supabase-schema.sql` — โครงฐานข้อมูล (รันแล้วบน Supabase)
- `supabase-schema-secure.sql` — migration ปิดการเข้าถึงตารางโดยตรง เปลี่ยนเป็นผ่านฟังก์ชันที่เช็ครหัส PIN ก่อนทุกครั้ง (รันแล้วบน Supabase)

ข้อมูลเก็บอยู่บน Supabase (ฐานข้อมูลกลาง) ทุกคนที่ล็อกอินจะเห็นข้อมูลร่วมกันแบบเรียลไทม์

**หมายเหตุความปลอดภัย:** anon key ในหน้าเว็บ (มองเห็นได้จาก view-source) ใช้เรียกได้เฉพาะฟังก์ชันที่เช็ค PIN ก่อนเสมอ — ไม่สามารถอ่าน/แก้ตารางข้อมูลตรงๆ ได้ แต่รหัส PIN 4 หลักยังไม่มีการจำกัดจำนวนครั้งที่ลองผิด (ไม่มี rate limit) จึงยังทายรหัสแบบ brute-force ได้ในทางทฤษฎี — เหมาะกับการใช้งานภายในทีมเล็กที่ไว้ใจกันได้ ไม่เหมาะกับข้อมูลที่ต้องการความปลอดภัยระดับสูง
